HTML Tags

The following are some commonly used HTML tags:

  • <div> - Defines a division or a section in an HTML document.
  • <p> - Defines a paragraph.
  • <a> - Defines a hyperlink.
  • <img> - Defines an image.
  • <ul> - Defines an unordered list.
  • <ol> - Defines an ordered list.
  • <li> - Defines a list item.
  • <table> - Defines a table.
  • <form> - Defines an HTML form for user input.
  • <input> - Defines an input control.

CSS Properties

Here are some CSS properties to style your HTML:

  • color - Specifies the text color.
  • font-size - Specifies the font size of text.
  • margin - Specifies the margin around an element.
  • padding - Specifies the padding inside an element.
  • background-color - Specifies the background color of an element.
  • border - Specifies the border of an element.
  • text-align - Specifies the horizontal alignment of text.
  • font-family - Specifies the font family for text.
